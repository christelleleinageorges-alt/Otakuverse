# User Stories

# Must have

- [x] L'utilisateur peut ajouter son numero afin d'accéder à l'application
- [x] L’utilisateur peut mettre des photos et videos sur l'application.
- [x] L’utilisateur peut ecrire des fanfictions sur l'application
- [x] L’utilisateur peut faire des appels videos ou normaux avec les personnes de son choix
- [x] L’utilisateur peut creer un groupe avec les personnes de son choix.
- [x] L’utilisateur peut mettre en favporis les sujets qui lui plaisent.
- [x] L'utilisateur peut se deconnecter de l'application


# Maybe

- [x] L’utilisateur peut bloquer les personnes de son choix,
- [ ] L’utilisateur peut modifier ou supprimer ses propres messages.
- [ ] L’utilisateur peut faire des appels en premier plan tout en partageant des histoires et des videos en arriere plan.
- [x] L’utilisateur peut avoir un profil personnel.
- [x] L’utilisateur peut modifier son profil personnel(Photo, Username, password]
- [ ] L'utilisateur peut creer un groupe.

# Would be nice to have
- [ ] L'utilisateur peut changer la langue(Creole --> Francais; Francais --> Creole).
- [ ] L'utilisateur peut changer le thème de l'appli(Light/Dark)
- [ ] L'utilisateur peut choisir les groupes de son choix.
- [ ] L’application peut mettre en favoris ce qu'il souhaite.
